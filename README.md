# Greenomy

커스터마이징 게임을 이용한 친환경 생활 실천 플랫폼 Greenomy

![팜플렛](https://user-images.githubusercontent.com/58512335/133219381-4bf74251-6ee1-4f5f-a934-7e07c065d09b.png)![팜플렛2](https://user-images.githubusercontent.com/58512335/133219383-56f67625-7a24-4db1-84f6-d979828d1683.png)




## :page_facing_up: Description
사람들이 환경보호 실천에 부담을 갖지 않고,  
누구나 쉽게 환경보호 실천에 참여할 수 있도록 환경보호 실천을 하면 포인트가 적립된다.  
적립된 포인트로 자신이 캐릭터 및 아이템을 구매해 자신만의 공간을 커스터마이징하고 친구들과 공유하며  
서로의 환경보호 생활에 동기부여를 더한다.  
이와 같은 온라인 친환경 게임 서비스 ‘Greenmoy’ 어플리케이션을 제공하고자 한다.
